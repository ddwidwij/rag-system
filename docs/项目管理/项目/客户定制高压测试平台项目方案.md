## 客户需求与背景

客户生产线主要测试 800V 平台 SiC 功率模块，现有测试方案存在以下不足：

- 测试电压上限为 600 V，无法覆盖 800 V 平台器件
- 测试通量不足，每天有效测试时间利用率低于 65%
- 缺乏自动分选功能，人工判级效率低且一致性差
- 高压环境下安全保护机制不完善，曾发生设备损坏事故

## 定制方案设计

### 硬件规格

| 参数 | 规格 |
|------|------|
| 最高测试电压 | 1200 V DC |
| 最大测试电流 | 50 A |
| 测试通道数 | 8 路并行 |
| 自动分选等级 | 5 级 |
| 安全保护 | 硬件互锁 + 软件双重保护 |
| 整机防护等级 | IP54 |

### 软件功能

- 测试配方管理：支持 SiC MOSFET、IGBT、SBD 等器件类型切换
- 数据追溯：批次追溯、测试历史查询、SPC 统计分析
- 远程诊断：支持远程升级和日志上传
- 报表导出：自动生成 CP 测试报告，兼容 Stdf/CSV 格式

## 风险与注意事项

1. 高压测试环境需客户提供符合 IEC 标准的测试间及接地系统
2. SiC 器件参数差异大，测试配方需在 FAT 阶段与客户联合标定
3. 分选等级划分需依据客户最新产品规格，变更需走 ECO 流程
