## 客户特殊需求记录

1. 客户要求所有软件版本、维保记录和异常分析报告均使用英文模板，并在最终版本中附客户可审计的签核页。
2. 由于跨境服务周期较长，客户要求关键易损件在本地保税仓维持 90 天安全库存，以降低等待时间。
3. 客户工厂网络与公司远程服务网络隔离，所有远程连接需要通过客户 IT 提前发放一次性 VPN 令牌。
4. 客户对 RF 测试模块的校准数据管理非常严格，要求每次校准均输出可追溯证书并同步到其 QMS 系统。

## 合作观察与后续建议

该客户重视国际化交付规范、备件响应和英文资料完整性。合作难点在于跨境沟通链条长、网络权限严格、现场窗口期短，因此所有版本发布、备件准备和现场服务计划都必须前置。后续建议维持双周项目例会，并固定输出英文周报和风险清单。
