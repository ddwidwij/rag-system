## 客户特殊需求记录

1. 实验室使用场景兼顾教学与科研，设备界面、教材和实验指导书需要提供更易理解的教学化版本。
2. 客户希望设备支持账户分级管理，以区分本科实验、研究生课题和外部联合项目的权限范围。
3. 所有演示数据和案例需脱敏处理，不得直接包含商业客户名称、量产参数或受控项目资料。
4. 联合课题期间形成的算法、脚本和实验结论需按合同约定区分知识产权归属，并保留共享边界。

## 合作观察与后续建议

该客户更看重教学体验、实验安全和联合研究扩展性，不以量产效率为首要目标。合作重点在于把设备能力转化为可教学、可演示、可复现实验流程，同时控制商业资料边界。后续建议持续补充实验教材、权限模板和脱敏案例库。
