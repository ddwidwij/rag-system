## 客户特殊需求记录

1. 客户对高压、耐温和重复性指标极为敏感，要求在交机时同步提供完整的量测不确定度与校准记录。
2. 试产期间的 recipe 需要按照封装类型和电压等级建立分层管理，防止不同器件族误用测试参数。
3. 客户现场要求 Chuck 温控范围覆盖 -55°C 至 +200°C，并在高温段给出稳定时间曲线作为验收依据。
4. 所有备件更换和参数调整都必须写入客户设备履历卡，便于后续与失效分析报告进行关联。

## 合作观察与后续建议

该客户虽然合作时间不长，但对高压功率器件测试的关注点非常集中，尤其强调高温稳定性和测试可追溯。后续项目推进中，应优先保证高压模块验证、温控曲线交付和 recipe 分层管理三项工作，帮助客户快速建立量产信心。
