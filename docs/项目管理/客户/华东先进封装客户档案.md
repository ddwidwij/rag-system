## 客户特殊需求记录

1. 客户试产阶段更关注换型效率和设备 OEE，要求在批量导入前给出不同封装形态的标准切换时间基线。
2. 现场使用的 MES 与条码系统由客户 IT 团队维护，所有接口变更需提前两周在集成测试环境完成联调。
3. 客户希望设备支持快速导出工位级良率报表，便于对比不同班次和不同夹具配置下的生产表现。
4. 由于厂区洁净规范严格，所有到场工程师必须提前完成 EHS 培训并取得临时施工授权。

## 合作观察与后续建议

该客户项目处于从试产向量产切换的关键阶段，对交付时效与数据联通要求较高。合作重点不是单纯交付设备，而是帮助客户建立稳定的换型、追溯和报表机制。后续应把封装工艺差异、MES 接口变更和驻场支持计划纳入固定月度清单，降低跨部门沟通成本。
