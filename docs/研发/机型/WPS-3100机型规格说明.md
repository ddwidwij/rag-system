# WPS-3100 机型规格说明

## 机型基本信息

| 属性 | 值 |
|------|----|
| 机型名称 | WPS-3100 高效率晶圆探针测试机 |
| 产品线 | 晶圆测试设备（WPS） |
| 机型代码 | WPS-3100 |
| 发布版本 | V1.3 |
| 发布日期 | 2025-05 |
| 适用晶圆尺寸 | 200 mm / 300 mm |

## 核心技术指标

- **测试通道数**：768 路数字通道 + 96 路模拟通道
- **最高测试频率**：2.5 GHz
- **换片时间**：< 6.5 秒
- **重复定位精度**：±0.4 µm
- **温控范围**：-50°C ~ +180°C
- **并行 Site 数**：最多 4 Site
- **平均可用率目标**：≥ 98%

## 主要子系统

1. **增强型测试头**：支持更高通道密度与模块化板卡扩展。
2. **高速运动平台**：优化长行程换片路径，缩短空程时间。
3. **升级版视觉模块**：支持弱纹理识别与 Bumped Wafer 对准。
4. **批次调度软件**：支持多 Site 并行和快速 Recipe 切换。

## 典型应用场景

- 高产能逻辑器件 CP 测试
- 多批次快速切换量产线
- 中高复杂度 SoC 晶圆参数测试
- 需要高节拍与高可用率的客户产线

## 选配模块

| 模块编码 | 模块名称 | 适用场景 |
|---------|---------|---------|
| OPT-MULTI | 多 Site 扩展模块 | 并行测试场景 |
| OPT-HV | 高压模块 | 功率器件与车规项目 |
| OPT-DATA | 高级追溯模块 | 需要丰富报表和数据分析的场景 |
