# WPS-3000 机型规格说明

## 机型基本信息

| 属性 | 值 |
|------|----|
| 机型名称 | WPS-3000 晶圆探针测试机 |
| 产品线 | 晶圆测试设备（WPS） |
| 机型代码 | WPS-3000 |
| 发布版本 | V2.1 |
| 发布日期 | 2025-03 |
| 适用晶圆尺寸 | 200 mm / 300 mm |

## 核心技术指标

- **测试通道数**：512 路数字通道 + 64 路模拟通道
- **最高测试频率**：2 GHz
- **测试精度（电压）**：±1 mV
- **测试精度（电流）**：±10 nA
- **温控范围**：-40°C ~ +175°C
- **晶圆对准精度**：±0.5 µm
- **标准换片时间**：< 8 秒

## 主要子系统

1. **测试头（Test Head）**：搭载 512 路 PE 板卡，支持热插拔维护。
2. **探针台（Prober）**：六轴运动控制，集成视觉对准与真空吸附。
3. **温控模块（Chuck）**：液氮/电阻丝双驱动，稳态精度可达 ±0.2°C。
4. **软件平台（WPS-OS）**：支持 STIL/ATML 测试格式与批次报表导出。

## 典型应用场景

- 逻辑芯片 CP 测试与良率筛选
- 存储器晶圆全温段参数筛选
- 功率器件耐压与导通电阻测试
- 射频芯片基础 S 参数测试

## 选配模块

| 模块编码 | 模块名称 | 适用场景 |
|---------|---------|---------|
| OPT-RF | 射频扩展模块 | 毫米波与高速接口芯片测试 |
| OPT-HV | 高压扩展模块 | 功率器件与耐压场景 |
| OPT-VI | 视觉增强模块 | Bumped Wafer 与弱纹理对准 |
