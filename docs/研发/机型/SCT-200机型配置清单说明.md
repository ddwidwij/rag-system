# SCT-200 机型配置清单说明

## 机型说明

SCT-200 为面向中小批量测试和研发验证场景的紧凑型测试平台，强调低占地、快部署和基础测试能力覆盖。该机型适合作为研发样机验证、实验线导入和中低复杂度器件测试平台使用。

## 标准配置

| 配置项 | 规格 |
|------|------|
| 数字测试通道 | 256 路 |
| 模拟测试通道 | 32 路 |
| 温控模块 | 常温 ~ 150°C |
| 视觉模块 | 基础对准相机 1 套 |
| 软件版本 | SCT-OS 1.2 |
| 机柜尺寸 | 1800 mm × 900 mm × 1650 mm |

## 标准随机附件

- 设备操作手册 1 份
- 基础维护工具包 1 套
- 备件包（滤芯、保险丝、常用连接件）1 套
- 安装与验收记录模板 1 套

## 可选配置

| 选配项 | 说明 |
|------|------|
| 高压模块 | 适用于耐压与功率器件测试 |
| 扩展视觉模块 | 提升对准精度与复杂晶圆识别能力 |
| 数据追溯包 | 增强批次和报表输出能力 |
| 客户定制接口板 | 对接特殊 DUT 或外部系统 |

## 配置使用建议

SCT-200 的优势在于部署成本低、上手快，但其通道数量、热能力和扩展性低于 WPS 主力机型。若项目预计进入高产能量产或高压复杂场景，应优先评估更高规格平台，而不是长期在 SCT-200 上叠加临时补丁。
