# WPS-3000 软件版本发布说明 V2.2

## 版本基本信息

|项目|内容|
|------|------|
|版本号|V2.2.0|
|发布日期|2025-06-08|
|适用机型|WPS-3000、WPS-3000Pro|
|前置版本|V2.1.0|
|发布类型|稳定性增强版本|

## 新增功能

### 1. 探针卡寿命监控
新增探针卡使用次数统计与阈值预警功能，达到 80 万次提醒、100 万次锁机，降低探针卡超期使用风险。

### 2. Recipe 差异比对
工程模式新增配方差异对比界面，可快速比较限值表、通道映射和时序配置，便于版本升级确认。

### 3. 批次级数据追溯增强
结果文件新增 Lot、Recipe 版本、操作员和软件版本字段，提升异常批次回溯效率。

## 缺陷修复

|缺陷编号|描述|严重等级|
|------|------|------|
|BUG-1108|多 Site 模式下个别 Site 状态未及时复位|严重|
|BUG-1126|Recipe 切换后日志未刷新当前版本号|一般|
|BUG-1141|工程模式导出报表时列顺序异常|一般|

## 升级注意事项

1. 升级前请导出当前探针卡与治具绑定配置，避免升级后映射丢失。
2. 首次启用寿命监控时需初始化历史计数，建议由现场工程师陪同执行。
3. 如果客户使用自定义报表模板，需同步更新字段解析脚本。
4. 回滚到 V2.1 前必须清理新增追溯字段缓存，避免旧版本读取异常。
