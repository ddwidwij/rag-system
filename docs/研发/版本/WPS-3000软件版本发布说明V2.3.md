# WPS-3000 软件版本发布说明 V2.3

## 版本基本信息

|项目|内容|
|------|------|
|版本号|V2.3.0|
|发布日期|2025-09-22|
|适用机型|WPS-3000、WPS-3000Pro|
|前置版本|V2.2.0|
|发布类型|功能增强版本|

## 新增功能

### 1. 高压测试安全联锁优化
重构高压通道上电前检查逻辑，增加绝缘自检、治具校验和安全门状态联动，降低误上电风险。

### 2. 并行报表导出
支持测试结束后并行生成 STDF、CSV 和客户自定义报表，缩短大批量批次出报表等待时间。

### 3. 异常批次快速复测
新增异常批次一键复测向导，可自动复用原配方和治具信息，减少人工重复配置。

## 缺陷修复

|缺陷编号|描述|严重等级|
|------|------|------|
|BUG-1180|高压场景切换治具后偶发安全状态未刷新|严重|
|BUG-1197|CSV 导出时中文字段在特定编码下乱码|一般|
|BUG-1204|复测模式下部分日志丢失批次标签|严重|

## 升级注意事项

1. 升级前需确认所有高压治具已完成绝缘检查并更新治具编号。
2. 若客户侧有旧版报表解析程序，请提前验证并行导出后的文件兼容性。
3. 升级完成后建议使用黄金样品执行至少一轮高压回归测试。
4. 异常批次复测向导会读取历史缓存，清理缓存后该功能将无法恢复旧上下文。
