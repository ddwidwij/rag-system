# WPS-3000 软件版本发布说明 V2.1

## 版本基本信息

|项目|内容|
|------|------|
|版本号|V2.1.0|
|发布日期|2025-03-15|
|适用机型|WPS-3000、WPS-3000Pro|
|前置版本|V2.0.3|
|发布类型|功能迭代版本|

## 新增功能

### 1. 多 Site 并行测试支持
新增最多 4 Site 并行测试能力，整机测试产能在标准逻辑器件场景下提升约 3.2 倍，需搭配 OPT-MULTI 扩展硬件模块使用。

### 2. 智能温度补偿算法
引入基于历史轨迹和实时温漂的补偿模型，在 -40°C 至 +175°C 全温区内将探针对准精度由 ±1.5 µm 提升至 ±0.5 µm。

### 3. 测试程序在线热更新
支持在不停机状态下替换 Pattern 与限值文件，减少换程序停机时间约 80%，适合多 Lot 快速切换场景。

## 升级注意事项

1. 升级前请备份 `/config/` 目录下所有配置文件和 Recipe 参数。
2. 本版本不兼容 V1.x 的 `.tpg` 测试程序格式，需提前转换后使用。
3. 升级后首次启动需执行完整系统自检，预计耗时 15 分钟。
4. 若现场存在客户定制接口，请先在验证机台完成回归测试后再切换量产机。
