# ATE-Link 平台版本升级说明 V3.0

## 版本基本信息

|项目|内容|
|------|------|
|版本号|V3.0.0|
|发布日期|2025-10-18|
|适用平台|ATE-Link 500、ATE-Link 300|
|前置版本|V2.7.4|
|发布类型|架构升级版本|

## 新增功能

### 1. 插件化驱动框架
底层驱动改为插件化装载方式，可按客户接口卡类型按需加载，减少主程序耦合。

### 2. 统一权限模型
工程模式、维护模式和操作模式权限统一接入平台账号体系，支持审计追踪。

### 3. 接口测试中心
新增外部系统联调中心，可集中验证 MES、QMS 和数据湖接口状态。

## 缺陷修复

|缺陷编号|描述|严重等级|
|------|------|------|
|BUG-5002|旧版驱动升级后个别接口卡识别超时|严重|
|BUG-5014|权限切换后日志未记录旧角色|一般|
|BUG-5021|接口中心导出结果时缺失部分错误码说明|一般|

## 升级注意事项

1. V3.0 属于架构升级版本，升级前必须完整备份驱动目录、权限配置和接口映射。
2. 所有第三方接口卡需要先验证对应插件包版本，不允许直接复用 V2.x 驱动。
3. 权限模型切换后需重新导入岗位与角色映射表。
4. 若客户现场为隔离网络环境，接口测试中心需使用离线模式部署。

## 版本兼容性矩阵
