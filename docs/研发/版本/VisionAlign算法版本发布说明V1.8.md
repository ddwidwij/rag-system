# VisionAlign 算法版本发布说明 V1.8

## 版本基本信息

|项目|内容|
|------|------|
|版本号|V1.8.0|
|发布日期|2025-07-02|
|适用模块|VisionAlign 视觉定位算法包|
|前置版本|V1.7.3|
|发布类型|算法增强版本|

## 新增功能

### 1. 弱纹理 Fiducial 识别增强
针对哑光表面和低对比度工艺片引入多尺度模板匹配与边缘增强策略，提高弱纹理场景识别成功率。

### 2. Bumped Wafer 特征过滤
增加 bump 反光噪声抑制逻辑，降低凸点晶圆在高亮环境下的误判率。

### 3. 识别质量评分输出
算法结果新增质量评分和失败原因标签，便于上层软件决定是否重拍或切换光源。

## 缺陷修复

|缺陷编号|描述|严重等级|
|------|------|------|
|BUG-VA-184|特定低照度场景下首帧定位偏移|严重|
|BUG-VA-191|质量评分在异常退出时未写入日志|一般|
|BUG-VA-196|Bump 场景下边缘过滤参数过敏|一般|

## 升级注意事项

1. 升级后建议重新训练或校验客户自定义模板库，避免旧模板与新评分逻辑不一致。
2. 质量评分接口为新增字段，上层软件未适配前应先关闭强依赖判断。
3. 若现场光源老化明显，建议同步进行光源均匀性检查，否则算法收益会被硬件瓶颈抵消。
4. 算法包回退到 V1.7 前请清理 V1.8 生成的评分缓存文件。
