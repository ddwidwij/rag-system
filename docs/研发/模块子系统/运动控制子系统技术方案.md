# 运动控制子系统技术方案

## 子系统概述

运动控制子系统（Motion Control Module，简称 MCM）是 WPS 系列设备实现高精度定位、高速换片和安全互锁的核心执行层，负责 X/Y 主平台、Theta 旋转轴、Z 轴升降、对刀机构和换片执行机构的协同联动。技术方案的核心目标不是单独追求轴速，而是在定位精度、路径平滑性、异常停车安全性和可维护性之间取得最优平衡。对于晶圆对准、探针接触、温控保温位置保持和故障恢复等关键场景，运动控制的稳定性直接决定整机测试节拍和误测风险。

## 模块架构

```
运动控制子系统
├── 运动控制器（Motion Controller）       — EtherCAT 主站、轨迹规划、多轴协调
├── 伺服驱动与编码器反馈链路             — 双编码器闭环、绝对值编码器
├── X/Y 主平台执行机构                   — 线性电机或丝杆+伺服、光栅尺测量
├── Theta 旋转轴                         — 高分辨率旋转台、对准角度补偿
├── Z 轴升降与接触保护机构               — 力位混合控制、软件/硬件限位双保护
└── 安全互锁与急停回路（Safety PLC）     — 门禁、限位、气压、真空、E-Stop 联锁
```

控制器通过 EtherCAT 与伺服驱动实时通讯，主控软件下发配方坐标、速度曲线和动作序列；安全 PLC 独立监控门禁、限位、气压和真空等互锁条件，满足 SIL-2 安全等级要求。发生异常时系统优先执行受控减速和安全回零，而非直接断伺服，防止惯性导致二次碰撞。

## 核心功能

### 1. 高精度定位与插补

X/Y 平台采用双编码器（电机侧+光栅尺）闭环反馈，Theta 轴使用高分辨率绝对值编码器。对准阶段执行短行程高精度插补，换片阶段切换为长行程高速模式。控制器根据工艺段自动切换 S 曲线加减速参数，减少高速转低速时的机械冲击，保护探针卡和晶圆表面。

### 2. Z 轴接触保护策略

Z 轴在接触阶段采用力位混合保护策略：接近晶圆表面时自动切换低速模式，实时读取接触力阈值与位移偏差；一旦超出上限立即停止下压并上报告警，禁止进入量产状态。该机制可有效减少探针针尖损伤和晶圆表面划伤，也是安规认证的强制要求项。

### 3. 异常检测与安全恢复

当出现限位触发、编码器丢脉冲、轴跟随误差超限等异常时，系统冻结当前动作序列并进行故障分类。可恢复故障允许在工程模式下执行重新回零；存在碰撞风险的故障必须人工复位与检查后方可恢复，系统不提供跳过安全检查的快捷通道。

## 关键参数

| 指标项 | 目标值 |
|------|------|
| X/Y 平台重复定位精度 | ±0.5 µm |
| Theta 轴重复定位精度 | ±0.002° |
| 最大换片移动速度 | 800 mm/s |
| Z 轴接触保护响应时间 | ≤ 10 ms |
| 运动异常停车距离 | ≤ 1.5 mm |
| EtherCAT 通讯周期 | 1 ms |

## 接口与维护要点

运动控制子系统与视觉检测子系统通过坐标补偿接口交互（对准结果驱动 XY 修正），与温控子系统共享 Chuck 位置状态（禁止未到温时执行接触动作），与安全 PLC 共享急停和互锁信号，与主控软件交换配方坐标、动作序列和故障码。维护时应重点关注以下事项：

- 每月检查丝杆、导轨和联轴器润滑状态，确认润滑周期执行到位并记录在履历中。
- 每季度复核回零偏差和编码器健康状态，若偏差趋势超出基线需提前介入排查。
- 发生碰撞或异常急停后，必须重新验证原点坐标、平面度和安全阈值，不允许仅复位后直接恢复量产。
- 不允许在未更新版本记录的情况下直接修改速度曲线、接触阈值和互锁参数。
- 联调阶段应优先确认坐标系定义、回零逻辑、限位方向和配方单位，避免模块间口径不一致引发碰撞。
