# 测试头板卡子系统维护指南

## 子系统概述

测试头板卡子系统（Test Head Board Module，简称 THB）是 WPS 系列设备直接与 DUT 交互的核心硬件单元，由数字通道板、模拟量测板、电源管理板、时序分配板和背板互连结构组成。该子系统长期处于高频开关、热负载变化和插拔维护并存的环境中，维护工作的核心目标是：保证信号完整性、供电稳定性、通道一致性，以及板卡更换后的快速恢复能力。对于量产机台，该子系统的健康状态直接决定良率数据可信度和客户交货节拍。

## 模块架构

```
测试头板卡子系统
├── 数字通道板（Digital Channel Board）  — 驱动/比较、时序控制、逻辑电平输出
├── 模拟量测板（Analog Measurement Board）— 电压/电流/阻抗/频率精密测量
├── 电源管理板（Power Supply Board）    — 多路 DUT 电源、精密偏置和保护回路
├── 时序分配板（Timing Distribution Board）— 多板同步时钟、触发信号分发
└── 背板互连结构（Backplane）           — 板间总线、电源轨道和信号屏蔽层
```

各功能板通过背板高速总线互连，数字通道板和模拟量测板在主控配方的调度下协同完成测试序列；电源管理板根据 DUT 规格独立控制各路偏置；时序分配板确保多板卡采样同步；背板兼顾信号完整性与散热结构。

## 核心功能

### 1. 日常巡检项目

外观与连接状态检查应重点关注板卡把手、锁紧机构、导风通道、背板连接器和屏蔽件是否完好。连接器氧化、导风口堵塞、固定螺钉松动或板卡轻微位移，均可能在高频测试阶段放大为通道不稳定问题。建议将巡检结果记录至设备履历，便于趋势分析。

### 2. 温度与电源状态监测

通过维护界面读取各板卡温度、供电电压、电流纹波和风扇状态。若某单板温升持续高于同型号均值 10°C 以上，应优先排查散热路径、功放负载和局部器件老化，而不是简单更换风扇。电源纹波超出规格时，应同步复核 DUT 负载和滤波器件状态。

### 3. 通道健康自检与校准

每次 PM 应执行通道开短路自检、参考负载回环测试和基准校准。量测漂移超出规格的通道，应记录板卡序列号、环境温度和最近一次软件升级信息，便于交叉分析漂移原因；不允许仅做复位处理后继续投入量产，未查明原因前需隔离该通道。

## 关键参数

| 参数 | 规格 |
|------|------|
| 数字通道数量（单机） | 最大 512 通道 |
| 模拟量测精度 | ±0.05%（25°C 基准） |
| 供电通道最大电流 | 4 A / 路 |
| 通道间串扰 | ≤ -60 dB（1 MHz） |
| 板卡工作温度范围 | 10°C ~ 50°C |
| PM 周期 | 每 500 小时或每月（取先到） |

## 接口与维护要点

测试头板卡子系统通过背板总线与数据采集子系统交互，通过配方接口接收主控软件的时序与限值配置，通过安全互锁接口与 E-Stop 和高压保护回路联动。维护时应重点关注以下事项：

- 停机维护前必须执行放电流程，确认高压和大电流通道已回到安全电位，再拔插板卡。
- 不同版本板卡禁止在未验证兼容性的情况下混插，板卡更换后须执行最小化自检和参考校准。
- 高压板卡和高频板卡应建立单独寿命台账，避免超龄运行导致绝缘失效。
- 现场若多次出现相同故障，需回溯软件版本、散热条件和负载工况，不允许仅做单板替换了事。
- 所有板卡更换、校准和返修操作必须形成可追溯记录，留存至设备履历。
