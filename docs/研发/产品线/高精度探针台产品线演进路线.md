# 高精度探针台产品线演进路线

## 产品线定位

高精度探针台（PRB 系列）面向半导体研发、失效分析和小批量验证场景，提供亚微米级步进精度的手动及半自动探针台，与 WPS 系列全自动机台形成高低端互补。

## 历史演进节点

| 年份 | 里程碑 | 关键技术突破 |
|------|--------|------------|
| 2018 | PRB-100 发布 | 首款自研探针台，步进精度 ±5μm |
| 2020 | PRB-200 发布 | 引入压电陶瓷精密驱动，步进精度 ±1μm |
| 2022 | PRB-300 发布 | 集成显微镜光路 + 电气测量一体化，支持 6 探针组 |
| 2023 | PRB-300T 发布 | 增加温控卡盘接口，与 TCS 系列兼容 |
| 2024 | PRB-400 立项 | 目标步进精度 ±0.1μm，支持 12" 晶圆 |

## 下一代 PRB-400 技术路线

**目标**：步进精度 ±0.1μm，对标进口竞品 Cascade/Suss

**关键技术攻关方向**：
1. **驱动系统**：压电陶瓷驱动器从 3 轴升级为 6 轴（X/Y/Z + Rx/Ry/Rz），实现倾斜补偿
2. **振动隔离**：从被动气浮台升级为主动隔振平台（AVI），抑制 1～100Hz 振动
3. **视觉对准**：引入亚微米级视觉对准系统，与 WPS-3000 共用算法模块
4. **软件控制**：统一集成到 ProberControl v5，支持脚本自动化测试流程

## 与 WPS 系列的差异定位

| 维度 | PRB 系列（高精度探针台） | WPS 系列（晶圆探针台） |
|------|----------------------|---------------------|
| 自动化程度 | 手动/半自动 | 全自动 |
| 主要场景 | 研发、FA、小批量 | 大规模量产测试 |
| 单片测试时间 | 10～60 分钟 | ≤8 秒/Die |
| 价格区间 | 15～80 万元 | 200～800 万元 |
| 客户类型 | 高校、设计院、研发中心 | 晶圆厂、封测厂 |

## 2025～2026 年关键里程碑

- **2025 Q2**：PRB-400 原型机完成，内部精度验证
- **2025 Q4**：PRB-400 Beta 机发往 3 家试用客户（高校科研 + 先进封装厂）
- **2026 Q2**：PRB-400 正式量产，目标首年销量 20 台
