# 岗位说明书：现场服务工程师

## 岗位基本信息

| 属性 | 内容 |
|------|------|
| 岗位编号 | POS-SVC-FE-001 |
| 岗位名称 | 现场服务工程师（Field Service Engineer） |
| 所属部门 | 服务部 |
| 直属上级 | 服务部区域经理 |
| 岗位级别 | 工程师级（P2~P4） |
| 工作地点 | 客户现场（长期驻外）|

## 岗位职责

1. **设备安装调试**：按照安装规范完成客户现场设备的开箱、安装、接线、调试和验收
2. **日常技术支持**：响应客户技术问题，现场排查和处理设备故障
3. **预防性维护（PM）**：按照 PM 计划执行定期保养，填写 PM 记录表
4. **工单管理**：准确填写服务工单，记录故障现象、处理过程和备件使用
5. **备件管理**：维护随车备件库存，及时补充消耗品
6. **客户培训**：向客户操作人员提供日常操作和基础维护培训
7. **信息反馈**：将现场发现的共性问题及时反馈给产品和研发团队

## 任职要求

### 学历与专业
- 本科及以上，电子工程、自动化、机电一体化或相关专业

### 工作经验
- P2：0~2 年，具备基本电路和机械知识，持有 WPS-3000 认证
- P3：3~5 年，具备独立处理严重故障能力，熟悉至少 2 个产品线
- P4：5 年以上，可担任 P2/P3 导师，具备客户关系管理能力

### 技能要求
- 熟练使用万用表、示波器等调试工具
- 能够阅读电路图、气路图和机械装配图
- 具备基本 Python/Shell 脚本能力（用于日志分析）
- 良好的客户沟通和服务意识

## 绩效考核维度

| 考核项 | 权重 | 目标值 |
|-------|------|-------|
| 工单及时关闭率 | 30% | ≥ 95%（严重故障 24h 关闭）|
| 客户满意度（CSAT） | 30% | ≥ 4.5 分（5 分制） |
| PM 完成率 | 20% | 100% |
| 认证维持 | 20% | 在有效期内持证上岗 |

## 职业发展路径

现场服务工程师 → 高级现场工程师 → 服务经理 / 技术专家
