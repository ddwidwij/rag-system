# 岗位说明书：质量工程师

## 岗位基本信息

| 属性 | 内容 |
|------|------|
| 岗位编号 | POS-QA-QE-001 |
| 岗位名称 | 质量工程师（Quality Engineer） |
| 所属部门 | 质量部 |
| 直属上级 | 质量部经理 |
| 岗位级别 | 工程师级（P3~P5） |
| 工作地点 | 公司本部（兼客户现场审核） |

## 岗位职责

1. **质量体系维护**：维护 ISO 9001 / IATF 16949 质量管理体系文件，组织内外部审核
2. **来料检验（IQC）**：制定关键物料检验规范，监督 IQC 执行，处理供应商质量问题
3. **过程质量控制（IPQC）**：建立生产过程控制计划（Control Plan），监控关键工序 Cpk
4. **出货检验（OQC）**：制定成品出货检验标准，确保出货质量符合客户要求
5. **客诉处理（8D）**：接收客户投诉，组织 8D 分析，制定纠正预防措施（CAPA），跟踪关闭
6. **失效分析协调**：协调 FA 实验室对失效样品进行根因分析，整合结论并输出 FA 报告
7. **供应商质量管理（SQM）**：对关键供应商进行定期审核和绩效评估

## 任职要求

### 学历与专业
- 本科及以上，材料、机械、电子工程或质量管理相关专业

### 工作经验
- P3：2~4 年，熟悉 QMS 文件体系，能独立处理客诉和 8D 报告
- P4：4~7 年，具备体系审核员资质（ISO 9001 内审员），能主导 CAPA 项目
- P5：7 年以上，可建立质量体系框架，具备 IATF 16949 外审经验

### 技能要求
- 熟练使用质量工具（FMEA、SPC、MSA、PPAP、8D）
- 熟悉半导体封测行业质量标准（AEC-Q100、JEDEC 等）
- 具备数据分析能力，会使用 Minitab 或 Excel 进行统计分析
- 持有 ISO 9001 内审员证书（P4 及以上要求）

## 绩效考核维度

| 考核项 | 权重 | 目标值 |
|-------|------|-------|
| 客诉 8D 按时关闭率 | 35% | ≥ 95%（10 个工作日内） |
| 审核发现不符合项整改率 | 25% | 100%（规定期限内） |
| 出货批次不良率（DPPM） | 25% | ≤ 50 DPPM |
| 供应商审核完成率 | 15% | 100%（按年度计划） |

## 职业发展路径

质量工程师 → 高级质量工程师 → 质量主管 / 质量体系经理
