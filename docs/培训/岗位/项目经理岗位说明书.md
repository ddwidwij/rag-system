# 岗位说明书：项目经理

## 岗位基本信息

| 属性 | 内容 |
|------|------|
| 岗位编号 | POS-PM-001 |
| 岗位名称 | 项目经理（Project Manager） |
| 所属部门 | 项目管理部（PMO） |
| 直属上级 | PMO 总监 |
| 岗位级别 | 经理级（M1~M2） |
| 工作地点 | 公司本部（兼客户现场） |

## 岗位职责

1. **项目计划制定**：根据合同要求和资源情况制定项目 WBS、进度计划和里程碑节点
2. **跨部门协调**：协调研发、测试、采购、服务等部门按计划推进，识别并消除阻塞点
3. **客户沟通**：作为客户主要联络人，定期汇报项目进展，管理客户期望
4. **风险管理**：识别项目风险，制定风险应对预案，跟踪风险状态至关闭
5. **变更管理**：评估范围变更的影响，走变更审批流程，更新项目基准
6. **预算管控**：监控项目实际成本，对偏差进行分析和汇报，控制预算在批准范围内
7. **项目收尾**：组织项目验收、客户签字，完成项目复盘，沉淀经验教训文档

## 任职要求

### 学历与专业
- 本科及以上，电子工程、工业工程、管理工程或相关专业

### 工作经验
- M1：3~6 年，具备独立管理中型项目（3~10 人团队，项目周期 6~18 个月）能力
- M2：6 年以上，具备多项目并行管理经验，能承担大型交付项目或战略客户项目

### 技能要求
- 熟练使用项目管理工具（MS Project、JIRA、飞书多维表格等）
- 掌握 PMP 项目管理方法论，具备 PMP 证书优先
- 具备扎实的技术背景，能理解半导体测试设备的交付复杂度
- 出色的跨部门协作、谈判和冲突管理能力

## 绩效考核维度

| 考核项 | 权重 | 目标值 |
|-------|------|-------|
| 项目按时交付率 | 35% | ≥ 90% |
| 客户验收一次通过率 | 25% | ≥ 85% |
| 项目预算偏差率 | 25% | ≤ ±5% |
| 项目复盘完成率 | 15% | 100%（每个项目必须复盘） |

## 职业发展路径

项目经理 → 高级项目经理 → PMO 经理 / 交付总监
