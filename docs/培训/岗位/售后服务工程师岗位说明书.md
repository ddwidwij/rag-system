# 岗位说明书：售后服务工程师

## 岗位基本信息

| 属性 | 内容 |
|------|------|
| 岗位编号 | POS-SVC-AS-001 |
| 岗位名称 | 售后服务工程师（After-Sales Service Engineer） |
| 所属部门 | 服务部 |
| 直属上级 | 服务部区域经理 |
| 岗位级别 | 工程师级（P2~P4） |
| 工作地点 | 公司驻地（兼部分出差） |

## 岗位职责

1. **远程技术支持**：通过电话、视频、远程桌面等方式响应客户技术问题，提供故障诊断指导
2. **备件申请与发货**：根据故障诊断结果为客户申请备件，跟踪备件发货及到货状态
3. **工单跟踪管理**：创建、更新和关闭服务工单，确保每笔工单有完整记录和责任闭环
4. **知识库维护**：将典型故障案例、处理方法整理入库，形成可复用的服务知识
5. **满意度跟进**：服务结束后对客户进行满意度回访，记录改进建议
6. **现场协调支持**：当远程无法解决时，协调现场工程师上门并提供技术背景材料
7. **保修期管理**：跟踪设备保修状态，区分保内/保外处理逻辑，协助合同续签

## 任职要求

### 学历与专业
- 本科及以上，电子工程、自动化、通信或相关专业

### 工作经验
- P2：0~2 年，能独立处理常见故障类型，熟悉主要产品工作原理
- P3：3~5 年，具备复杂故障远程排查能力，熟悉至少 2 个产品线售后流程
- P4：5 年以上，可制定服务流程规范，承担新品售后体系搭建

### 技能要求
- 熟悉 ATE 设备、半导体测试系统基本工作原理
- 能通过日志文件分析定位软硬件故障根因
- 具备良好的文档撰写能力（工单、故障报告、服务简报）
- 出色的客户沟通与情绪管理能力

## 绩效考核维度

| 考核项 | 权重 | 目标值 |
|-------|------|-------|
| 工单首次响应时效 | 30% | ≤ 2 小时（工作时间内） |
| 客户满意度（CSAT） | 30% | ≥ 4.5 分（5 分制） |
| 工单远程解决率 | 20% | ≥ 60% |
| 知识库贡献条目数 | 20% | ≥ 4 条/季度 |

## 职业发展路径

售后服务工程师 → 高级售后工程师 → 售后服务主管 / 技术支持专家
