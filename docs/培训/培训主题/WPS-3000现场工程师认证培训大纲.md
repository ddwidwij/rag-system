# 培训课程：WPS-3000 现场工程师认证培训

## 课程基本信息

| 属性 | 内容 |
|------|------|
| 培训主题编号 | TRN-FE-WPS3K-001 |
| 培训主题 | WPS-3000 现场工程师认证培训 |
| 适用岗位 | 现场服务工程师、安装调试工程师 |
| 培训方式 | 线上理论（16h）+ 线下实操（24h） |
| 认证有效期 | 2 年 |
| 培训负责人 | 培训中心 |

## 培训目标

完成本培训后，学员应具备以下能力：
1. 独立完成 WPS-3000 的安装、调试和验收
2. 正确使用 WPS-OS 软件执行测试和故障诊断
3. 处理常见故障码（VIS / MOT / TCU 系列）
4. 执行一级预防性维护（PM1）操作
5. 向客户提供规范的技术支持服务

## 培训大纲

### 模块一：产品认知（4h）
- WPS-3000 产品线定位与技术规格
- 主要子系统介绍（视觉、运动、温控）
- 选配模块说明

### 模块二：软件操作（6h）
- WPS-OS 界面与基本操作
- 测试程序加载与执行
- 数据查看与 STDF 文件解析
- 系统配置与参数设置

### 模块三：安装调试（8h）
- 机台开箱与安装流程
- 水电气接入规范
- 首次上电与系统自检
- 视觉标定与运动原点设置

### 模块四：故障处理（10h）
- 故障码体系介绍
- VIS/MOT/TCU 常见故障排查
- 故障案例分析（含 VIS-1002 案例）
- 工单填写与升级规范

### 模块五：预防维护（6h）
- PM1 标准作业程序（SOP）
- 消耗品更换（探针卡、滤网、润滑脂）
- 设备状态记录与上报

### 模块六：认证考核（6h）
- 理论笔试（60 分钟，合格线 80 分）
- 实操考核（独立处理 3 个模拟故障场景）

## 考核标准

| 考核项 | 权重 | 合格线 |
|-------|------|-------|
| 理论笔试 | 40% | ≥ 80 分 |
| 实操考核 | 60% | 3 个场景全部通过 |

综合得分 ≥ 80 分且实操全通过，颁发认证证书。
