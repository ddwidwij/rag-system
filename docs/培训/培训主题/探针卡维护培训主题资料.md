# 培训课程：探针卡维护培训

## 课程基本信息

| 属性 | 内容 |
|------|------|
| 培训主题编号 | TRN-MT-PRB-001 |
| 培训主题 | 探针卡日常维护与寿命管理培训 |
| 适用岗位 | 现场服务工程师、工艺工程师 |
| 培训方式 | 理论授课（4h）+ 实操演练（8h）|
| 认证有效期 | 2 年 |
| 培训负责人 | 工艺支持部 |

## 培训目标

完成本培训后，学员应具备以下能力：
1. 识别探针卡类型（标准卡、射频卡、高压卡）并正确选型
2. 规范执行探针卡安装、拆卸和清洁操作
3. 判断探针卡磨损状态，执行针尖研磨或更换决策
4. 正确记录探针卡使用次数并管理寿命台账
5. 处理探针卡相关测试异常（接触电阻超标、良率降低）

## 培训大纲

### 模块一：探针卡基础知识（2h）
- 探针卡组成结构（针尖、固定座、PCB 载板、空间变换器）
- 主要料号介绍（PRB-C01/C02/C03/C04）
- 不同类型探针卡适用场景
- 探针卡性能参数（针压、接触电阻、绝缘阻抗）

### 模块二：安装与拆卸规范（3h）
- 探针卡安装前检查（外观、针尖状态、PCB 清洁度）
- 安装对准操作（CCD 辅助对准步骤）
- 针压调整与验证方法
- 拆卸保存规范（防静电包装、针尖保护盖）

### 模块三：日常清洁与维护（2h）
- 针尖清洁周期与清洁材料选择（探针清洁片、氧化铝研磨片）
- 电气测量验证（接触电阻 ≤ 3 Ω 标准）
- 外观检查要点（弯针、断针、针座氧化）
- 异常状态上报流程

### 模块四：寿命管理与更换决策（2h）
- 探针卡寿命统计（台账模板说明）
- 寿命到期预警（建议更换周期：钨针 100 万次，Pogo Pin 50 万次）
- 针尖研磨 vs 整卡更换的决策树
- 旧探针卡回收与处置规范

### 模块五：实操演练（8h）
- 拆装探针卡全流程操作
- 针尖研磨实操（使用研磨片，目标：接触电阻恢复至 ≤ 2 Ω）
- 模拟接触电阻超标故障排查

## 考核标准

| 考核项 | 权重 | 合格线 |
|-------|------|-------|
| 理论测试 | 30% | ≥ 75 分 |
| 实操考核（含安装+清洁+排障）| 70% | 全部步骤合格 |

综合得分 ≥ 80 分，颁发探针卡维护操作认证资质。
