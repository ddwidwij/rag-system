# 培训课程：WPS-3200 新员工培训

## 课程基本信息

| 属性 | 内容 |
|------|------|
| 培训主题编号 | TRN-NE-WPS32-001 |
| 培训主题 | WPS-3200 新员工入职培训 |
| 适用岗位 | 研发工程师、测试工程师、产品支持工程师 |
| 培训方式 | 线上理论（12h）+ 线下参观实操（8h）|
| 认证有效期 | 不设期限（入职必修）|
| 培训负责人 | 人才发展部 |

## 培训目标

完成本培训后，学员应具备以下能力：
1. 了解 WPS-3200 的产品定位、主要技术规格和目标客户
2. 掌握 WPS-3200 与 WPS-3000Pro 的核心差异
3. 理解 WPS-3200 各子系统（视觉、运动、温控、测试头）的基本原理
4. 能够规范使用内部文档系统查阅技术资料
5. 了解新员工在本岗位的工作流程与协作规范

## 培训大纲

### 模块一：公司与产品线概览（2h）
- 公司业务版图与战略方向
- 晶圆测试设备行业背景
- WPS 产品线全貌（WPS-3000/3000Pro/3200 演进路线）

### 模块二：WPS-3200 技术概述（4h）
- WPS-3200 主要技术规格（2048 通道、8 GHz、液冷、±0.1 µm）
- 关键子系统架构介绍
- WPS-3200 vs WPS-3000Pro 差异对比
- 典型应用场景与目标客群

### 模块三：子系统原理入门（4h）
- 视觉对准子系统工作原理
- XY/Z/θ 运动平台机械结构
- 温控子系统（TEC 制冷 + 液冷循环）
- 测试头与探针卡基础知识

### 模块四：研发流程与文档规范（2h）
- 产品研发阶段划分（M0~M6）
- 设计文档体系与编号规范
- 内部知识库使用方法
- ECO/CR 变更流程简介

### 模块五：实地参观与上机体验（8h）
- 参观 WPS-3200 样机及装配车间
- 上机观摩测试流程演示
- 与各部门负责人 Q&A 交流

## 考核标准

| 考核项 | 权重 | 合格线 |
|-------|------|-------|
| 线上理论测试 | 60% | ≥ 75 分 |
| 参观实习报告 | 40% | 提交完整报告 |

综合得分 ≥ 75 分，完成入职培训认证，进入岗位专项培训阶段。
