# 培训课程：质量闭环与 8D 方法培训

## 课程基本信息

| 属性 | 内容 |
|------|------|
| 培训主题编号 | TRN-QA-8D-001 |
| 培训主题 | 质量闭环管理与 8D 问题解决方法 |
| 适用岗位 | 质量工程师、研发工程师、服务工程师、项目经理 |
| 培训方式 | 理论授课（8h）+ 案例研讨（4h）|
| 认证有效期 | 2 年 |
| 培训负责人 | 质量管理部 |

## 培训目标

完成本培训后，学员应具备以下能力：
1. 掌握 8D 报告的八个步骤并能独立撰写
2. 使用 5 Why 和鱼骨图工具找到真正根本原因
3. 制定有效的纠正措施（CA）和预防措施（PA）
4. 按规范管理 CAPA 条目和闭环时间节点
5. 识别并避免常见的 8D 报告质量缺陷

## 培训大纲

### 模块一：质量问题管理体系概述（2h）
- 质量问题来源分类（客户投诉、内部审计、生产异常）
- 质量问题严重等级定义（S1~S4）
- 质量闭环管理流程（发现 → 遏制 → 分析 → 改善 → 预防 → 验证 → 关闭）
- 相关系统使用说明（CAPA 管理系统操作介绍）

### 模块二：8D 方法详解（4h）
- D1：组建问题解决团队（跨职能、明确职责）
- D2：问题描述（5W2H，量化表述）
- D3：临时遏制措施（立即止损，不影响根因分析）
- D4：根本原因分析（5 Why + 鱼骨图 + 验证方法）
- D5：长期纠正措施（CA 方案选择标准）
- D6：实施并验证纠正措施（效果数据支撑）
- D7：预防措施（扩展到类似问题和产品）
- D8：团队认可与问题关闭（经验沉淀入库）

### 模块三：常见质量问题案例研讨（4h）
- 案例一：WPS-3000 视觉对准重复性不良（VIS-1002 类问题）
  - 问题描述 → 根因分析 → 纠正措施 → 预防扩展
- 案例二：探针卡更换后接触电阻超标导致客户投诉
  - 临时遏制 → 5 Why 分析 → SOP 修订 → 培训推广
- 案例三：版本升级后生产良率下降
  - 问题定界 → 变更关联分析 → 回滚决策 → 长期改善

### 模块四：8D 报告质量评审（2h）
- 评审维度：问题描述清晰度、根因深度、措施有效性、预防覆盖面
- 常见质量缺陷（根因停在"操作失误"而未追溯系统原因）
- 优秀 8D 报告示例赏析

## 考核标准

| 考核项 | 权重 | 合格线 |
|-------|------|-------|
| 理论测试 | 30% | ≥ 80 分 |
| 8D 报告撰写（给定案例）| 70% | 评审得分 ≥ 80 分 |

综合得分 ≥ 80 分，颁发 8D 方法认证资质。
