# 温控模块验收测试用例集

## 测试用例基本信息

| 属性 | 内容 |
|------|------|
| 用例集编号 | TC-TEMP-VAL-001 |
| 所属测试项 | 温控模块出厂验收测试 |
| 适用产品 | WPS 系列温控附件（Chiller/Heater 模块） |
| 编写人 | 测试工程团队 |
| 版本 | V2.0 |

## 测试用例列表

### TC-001：目标温度达到时间验证
- **前置条件**：温控模块冷却液初始温度为 25°C，设置目标温度 −40°C
- **输入**：下发温度设定指令，启动制冷
- **预期输出**：探针台卡盘实测温度在 15 分钟内降至 −40°C ± 1°C
- **判定**：Pass（超时则 Fail）
- **优先级**：P1

### TC-002：温度稳定性验证（长时保温）
- **前置条件**：温控模块已将卡盘温度稳定在 −40°C
- **输入**：保持温度设定不变，持续运行 2 小时
- **预期输出**：2 小时内卡盘实测温度波动范围 ≤ ±0.5°C
- **判定**：Pass
- **优先级**：P1

### TC-003：超温保护触发测试
- **前置条件**：温控模块正在加热至 150°C
- **输入**：模拟温度传感器故障，返回 200°C 异常数值
- **预期输出**：模块立即切断加热，触发 Alarm-TEMP01（超温保护），系统进入安全冷却模式
- **判定**：Pass（保护响应时间 ≤ 500 ms）
- **优先级**：P1

### TC-004：温度跳变抗扰性测试
- **前置条件**：卡盘温度稳定在 25°C
- **输入**：快速切换目标温度，−40°C → 125°C → −40°C（连续 3 次）
- **预期输出**：每次切换后均能在规定时间内达到目标温度，无过冲超过 ±3°C
- **判定**：Pass
- **优先级**：P2

### TC-005：冷媒泄漏报警验证
- **前置条件**：温控模块正常运行
- **输入**：人工触发泄漏传感器模拟信号
- **预期输出**：系统立即停止制冷，触发 Alarm-TEMP02（冷媒泄漏），同时向 EAP 上报告警事件
- **判定**：Pass
- **优先级**：P2

## 回归测试要求

温控模块固件升级或控温算法（PID 参数）调整后，TC-001 至 TC-003 为必须回归用例；TC-004 和 TC-005 在传感器驱动或告警逻辑有变更时纳入回归。
