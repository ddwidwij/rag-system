# 运动控制回归测试用例集

## 测试用例基本信息

| 属性 | 内容 |
|------|------|
| 用例集编号 | TC-MOT-REG-001 |
| 所属测试项 | 探针台运动控制系统回归测试 |
| 适用产品 | WPS-3000 XYZ 运动平台 |
| 编写人 | 测试工程团队 |
| 版本 | V1.4 |

## 测试用例列表

### TC-001：回零动作正确性验证
- **前置条件**：XYZ 三轴处于任意非零位置，系统已上电完成自检
- **输入**：执行"回零"指令
- **预期输出**：三轴依次运动至零位传感器触发点，坐标系清零，显示 X=0.000, Y=0.000, Z=0.000（单位 µm）
- **判定**：Pass（偏差 > ±0.5 µm 为 Fail）
- **优先级**：P1

### TC-002：定点移动精度验证
- **前置条件**：系统回零完成，使用已标定的精密标靶板
- **输入**：命令 X 轴移动至 50000 µm，Y 轴移动至 30000 µm
- **预期输出**：视觉系统实测落点坐标与命令坐标偏差 ≤ ±1 µm（3σ）
- **判定**：Pass
- **优先级**：P1

### TC-003：软限位触发保护测试
- **前置条件**：X 轴当前位置距右侧软限位 100 µm
- **输入**：发送超出软限位范围的移动指令（目标超限 500 µm）
- **预期输出**：运动控制器拒绝执行，返回错误码 MOT-ERR-03（软限位越界），轴停止在当前位置
- **判定**：Pass
- **优先级**：P1

### TC-004：连续步进精度退化测试
- **前置条件**：系统回零完成
- **输入**：在 X 方向以 5 mm 步长连续移动 200 次（往返），视觉测量每次落点
- **预期输出**：200 次步进后累计定位误差 ≤ ±2 µm，无异常抖动或跳步
- **判定**：Pass
- **优先级**：P2

### TC-005：急停恢复后位置保持验证
- **前置条件**：Z 轴正在下针过程中（接触高度前 50 µm）
- **输入**：触发急停（E-Stop）
- **预期输出**：Z 轴立即停止（响应时间 ≤ 100 ms），急停后位置读数与停止前最后记录值一致，误差 ≤ ±0.5 µm；重新上电后系统拒绝自动继续下针，需人工确认
- **判定**：Pass
- **优先级**：P2

## 回归测试要求

运动控制固件更新、编码器更换或伺服参数（增益/加速度）调整后，TC-001 至 TC-003 为必须回归用例；TC-004 在长时间运行精度相关模块变更时纳入；TC-005 在急停逻辑或上电恢复流程有变更时纳入回归。
