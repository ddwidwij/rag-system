# RF前端校准测试项标准

## 文档概述

本文档定义射频（RF）前端芯片在晶圆级与封装级测试中的校准测试项（Calibration Test Item），包括测试项编码、校准方法、判定标准和仪器要求，适用于集成 PA、LNA、混频器等功能模块的 RF SoC 产品。

## 测试项分类

RF 前端校准测试项分为功率校准、频率校准和线性度校准三大类：

| 分类 | 缩写 | 典型测试项数量 |
|------|------|------------|
| 功率校准 | PWR | 10~30 项 |
| 频率校准 | FREQ | 5~15 项 |
| 线性度校准 | LIN | 8~20 项 |

## 常用测试项说明

### PWR-001：发射功率校准（TX Power Cal）
- **目的**：确认发射链路输出功率符合设计目标
- **方法**：配置增益寄存器至标称值，测量射频输出端功率
- **判定**：功率偏差 > ±1.5 dBm 为 Fail
- **仪器条件**：频谱仪或功率计，负载阻抗 50Ω

### PWR-002：接收灵敏度校准（RX Sensitivity Cal）
- **目的**：验证接收链路在最低输入信号下的检测能力
- **方法**：注入参考信号，调整 LNA 增益，使输出 SNR ≥ 10 dB
- **判定**：灵敏度劣于规格 3 dB 以上为 Fail
- **温度条件**：25°C 校准，−40°C 及 85°C 验证

### FREQ-001：中心频率偏差校准（Freq Offset Cal）
- **目的**：修正晶振偏差引起的载波频率漂移
- **方法**：测量实际载波频率与标称频率之差，写入校准寄存器
- **判定**：校准后频率偏差 > ±10 ppm 为 Fail
- **执行时机**：封装后首次上电强制执行

### LIN-001：IQ 不平衡校准（IQ Imbalance Cal）
- **目的**：消除正交调制路径的幅度/相位不一致
- **方法**：发射 CW 信号，测量镜像抑制比（IRR），写入补偿系数
- **判定**：校准后 IRR < 30 dBc 为 Fail

### LIN-002：三阶截点测试（IIP3）
- **目的**：评估接收链路的非线性互调抑制能力
- **方法**：注入双音信号（Δf = 1 MHz），测量三阶交调产物
- **判定**：IIP3 < −5 dBm 为 Fail
- **频率条件**：在工作频段最低、中间、最高频点各测一次

## 测试项顺序规范

建议按照"功率校准 → 频率校准 → 线性度校准"顺序执行。功率链路异常时频率和线性度结果无参考意义，应提前 Fail Bin，避免无效校准写入，缩短平均测试时间（ATT）。
