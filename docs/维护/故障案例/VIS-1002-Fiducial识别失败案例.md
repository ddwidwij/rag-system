# 故障案例：VIS-1002 Fiducial 识别失败

## 案例基本信息

| 属性 | 内容 |
|------|------|
| 案例编号 | CASE-2025-VIS-001 |
| 故障码 | VIS-1002 |
| 机型 | WPS-3000（SN: WPS3K-2024-0087） |
| 发生日期 | 2025-01-12 |
| 客户 | XX 半导体（苏州）有限公司 |
| 严重等级 | 严重 |
| 处理工程师 | 李工 |

## 故障现象

设备在换片后执行自动对准时，连续报出 VIS-1002 故障，无法完成 Fiducial Mark 识别，晶圆测试中断。故障发生频率：每 3~5 片换片后必现。

## 排查过程

**步骤一**：检查相机图像质量
打开视觉调试界面，实时查看底部相机画面，发现图像整体偏暗，对比度低，但无明显噪点。

**步骤二**：检查光源状态
调用光源控制面板，切换同轴光源与环形光源。环形光源在低亮档位下晶圆表面反射不足，导致 Fiducial 识别阈值不够。

**步骤三**：确认问题根因
该批次晶圆表面为哑光氮化硅钝化层，反射率比标准晶圆低约 40%，原有光源亮度参数不适用。

## 解决方案

1. 将底部相机环形光源亮度参数从 `60%` 调整至 `85%`
2. 将视觉识别对比度阈值从 `0.65` 降低至 `0.50`
3. 在测试程序 `recipe` 文件中新增晶圆类型标识，实现按晶圆类型自动切换光源参数

## 验证结果

调整后连续测试 200 片晶圆，VIS-1002 故障未再出现，Fiducial 识别成功率 100%。

## 经验沉淀

对于表面特殊工艺（哑光、厚氧化层、铜面）的晶圆，首次测试前需在视觉调试模式下验证光源参数，将参数记录至机台工艺配置档案。
